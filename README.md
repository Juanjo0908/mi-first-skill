# 📊 Titan ML Logic - CSV Analyzer

Este proyecto es una **Skill de Inteligencia Artificial** construida con Python y Flask. Su objetivo es procesar archivos CSV para extraer métricas clave y preparar datos para modelos de Machine Learning.

## 🚀 Características
- **Análisis Automatizado:** Obtiene dimensiones, tipos de datos y valores nulos al instante.
- **Preparado para ML:** Diseñado para integrarse con flujos de trabajo de ciencia de datos.
- **Arquitectura Serverless:** Optimizado para correr en plataformas como Vercel o Skills.sh.

## 🛠️ Tecnologías utilizadas
* **Python 3.12+**
* **Flask:** Para la interfaz de la API.
* **Pandas:** Para la manipulación y análisis de datos.
* **Gunicorn:** Servidor WSGI para producción.
